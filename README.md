MajuPay - E-Wallet Professional (DANA Inspired)
  

MajuPay adalah aplikasi dompet digital berbasis web yang dikembangkan menggunakan React 18, Tailwind CSS, dan Bootstrap 5. Proyek ini dirancang untuk mensimulasikan antarmuka dan alur kerja aplikasi finansial modern dengan standar industri UI/UX.  

⚠️ PENTING: CATATAN KOMPATIBILITAS

  

Aplikasi ini dirancang dengan pendekatan Mobile-Only. Layout tidak responsif untuk tampilan desktop lebar dan hanya dioptimalkan untuk rasio aspek perangkat mobile. Sangat disarankan untuk membukanya menggunakan fitur Inspect Element (Device Mode) pada browser dan memilih perangkat seperti iPhone atau Pixel.  

Fitur Utama
  

Single Page Application (SPA): Navigasi cepat tanpa reload menggunakan React-Router.  

State Management: Update saldo secara real-time di seluruh komponen aplikasi menggunakan Context API.  

Alur Transaksi Lengkap: Simulasi pengiriman uang dari pemilihan metode hingga konfirmasi PIN.  

Professional UI: Menggunakan komponen Bootstrap 5 untuk struktur data dan Tailwind CSS untuk desain visual yang presisi.  

Static PIN Pad: Keamanan transaksi dengan input PIN standar (0-9) yang stabil.
